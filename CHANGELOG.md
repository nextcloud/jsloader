# Changelog

## v2.1.0 - 2026-05-21
### Added
- Added compatibility with Nextcloud 32, Nextcloud 33, and Nextcloud 34.
  - Dropped compatibility with Nextcloud 29 as minimum PHP version was bumped to 8.1.

### Changed
- Migrated the frontend to pure JavaScript for future Nextcloud compatibility.
- Align design of the frontend with Nextcloud 30+ design changes.

## v2.0.0 - 2025-04-09
### Added
- Added compatibility for Nextcloud 29, 30, and 31.
